## Description

Calculates the remainder of two values and stores the result.

## Pseudo Code

```cpp

    const auto val2 = FUNC_XiEvent_getworkofs_(this, 5);

    auto res = 0;
    if (val1 && val2)
        res = val1 % val2;
    else
        res = 0;

    FUNC_XiEvent_setworkofs_(this, 1, res);

    this->ExecPointer += 7;
}
```
